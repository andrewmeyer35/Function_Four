# Roadmap

## Phase 0 — Concept Test (Weeks 1–2)
- [ ] Run manual check-in experiment with roommates
- [ ] Simple Google Form (one field per F), share in group chat each Sunday
- [ ] Observe: who comes back week 2? Which F gets skipped most?

## Phase 1 — Web MVP (Months 1–2)
### Must Have
- [ ] Auth — email magic link via Supabase
- [ ] Weekly check-in form (one text field per F)
- [ ] Household feed — reverse chrono, grouped by week
- [ ] Emoji reactions on check-ins
- [ ] Friday reminder email (Resend)
- [ ] Household invite via shareable link + code

### Nice to Have
- [ ] Streak counter (consecutive weeks checked in)
- [ ] "Still waiting on..." banner for missing housemates

## Phase 2 — Multi-Household (Months 3–5)
- [ ] Crews — named groups spanning multiple households
- [ ] Cross-household feed filterable by F category
- [ ] Event proposals (runs, dinners, nights out)
- [ ] RSVP + event chat thread
- [ ] Leaderboard / streaks across crew

## Phase 3 — iOS App (Months 6–9)
- [ ] React Native / Expo build (reuses frontend logic)
- [ ] Native push notifications (Expo Push)
- [ ] Home screen widget — crew weekly status at a glance
- [ ] App Store submission
- [ ] Premium tier: larger crews, analytics, custom F categories
